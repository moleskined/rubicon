Rubicon — a Caesar cipher plaintext solver

This is a simple utility to aid in finding the plaintext from a given ciphertext and identify the shift-by-n value used to encrypt the original plaintext.

A Caesar cipher is a type of substitution cipher in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. (‘Caesar cipher - Wikipedia’ n.d.)
